Module lib::assert_some::assert_some_ne

Expand description

Assert two expressions are Some and their values are not equal.

Pseudocode:
(a ⇒ Some(a1) ⇒ a1) ≠ (b ⇒ Some(b1) ⇒ b1)

§Example

use assertables::*;

let a: Option<i8> = Option::Some(1);
let b: Option<i8> = Option::Some(2);
assert_some_ne!(a, b);
