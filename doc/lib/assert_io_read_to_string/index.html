Assert for comparing input/output reader streams.

These macros help with input/output readers, such as file handles, byte arrays, input streams, the trait ::std::io::Read, and anything that implements a method read_to_string() -> String.

Compare a reader with another reader:

  • assert_io_read_to_string_eq2!(reader1, reader2) ≈ reader1.read_to_string() = reader2.read_to_string()

  • assert_io_read_to_string_ne2!(reader1, reader2) ≈ reader1.read_to_string() ≠ reader2.read_to_string()

  • assert_io_read_to_string_lt2!(reader1, reader2) ≈ reader1.read_to_string() < reader2.read_to_string()

  • assert_io_read_to_string_le2!(reader1, reader2) ≈ reader1.read_to_string() ≤ reader2.read_to_string()

  • assert_io_read_to_string_gt2!(reader1, reader2) ≈ reader1.read_to_string() > reader2.read_to_string()

  • assert_io_read_to_string_ge2!(reader1, reader2) ≈ reader1.read_to_string() ≥ reader2.read_to_string()

Compare a reader with an expression:

  • assert_io_read_to_string_eq!(reader, expr) ≈ reader.read_to_string() = expr

  • assert_io_read_to_string_ne!(reader, expr) ≈ reader.read_to_string() ≠ expr

  • assert_io_read_to_string_lt!(reader, expr) ≈ reader.read_to_string() < expr

  • assert_io_read_to_string_le!(reader, expr) ≈ reader.read_to_string() ≤ expr

  • assert_io_read_to_string_gt!(reader, expr) ≈ reader.read_to_string() > expr

  • assert_io_read_to_string_ge!(reader, expr) ≈ reader.read_to_string() ≥ expr

Compare a reader with its contents:

  • assert_io_read_to_string_contains!(reader, &containee) ≈ reader.read_to_string().contains(containee)

  • assert_io_read_to_string_is_match!(reader, &matcher) ≈ matcher.is_match(reader.read_to_string())

§Example

use assertables::*;
use std::io::Read;

let mut a = "alfa".as_bytes();
let mut b = "alfa".as_bytes();
assert_io_read_to_string_eq2!(a, b);
